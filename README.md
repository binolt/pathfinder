## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/binolt/pathfinder.git
   ```
2. Install dependencies
   ```sh
   npm install
   ```
3. Start the live server
   ```sh
   npm run dev
   ```
